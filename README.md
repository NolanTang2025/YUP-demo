# YUP信用卡用户意图聚类分析系统

## 📋 项目简介

本项目针对YUP信用卡公司客户获得额度后的第一次交易行为进行深度分析，通过机器学习聚类算法识别不同用户的意图类型，并提供专业的可视化分析报告。

## 🎯 分析目标

- **用户意图识别**：区分完成交易用户和未完成交易用户的行为模式
- **行为特征提取**：从用户行为数据中提取多维特征
- **聚类分析**：使用K-means算法对用户进行意图聚类
- **可视化展示**：生成专业的HTML分析报告

## 📊 数据说明

数据文件：`data.xlsx`

包含字段：
- `user_uuid`: 用户唯一标识
- `act_time`: 账户激活时间
- `fir_trx_time`: 首次交易时间（如果完成交易）
- `event_time`: 事件发生时间
- `event_name`: 事件名称
- `remarks`: 备注信息

## 🔍 特征工程

系统提取了以下用户行为特征：

### 基础特征
- 总事件数
- 会话时长
- 唯一事件类型数
- 事件密度（事件/分钟）

### 时间特征
- 平均事件间隔时间
- 事件间隔时间标准差
- 最大事件间隔时间

### 行为模式特征
- 探索得分：衡量用户探索不同功能的程度
- 执行得分：衡量用户执行交易行为的倾向
- 重复行为得分：识别用户犹豫或反复查看的行为
- 行为转换次数：用户在不同功能间切换的频率

### 事件类型特征
- 主页访问次数
- 优惠券交互次数
- QR码交互次数
- 支付尝试次数
- 个人资料查看次数

## 📈 分析结果

### 聚类结果

系统识别出两种用户意图类型：

1. **高转化意图用户（Cluster 1）**
   - 特征：成功完成首笔交易
   - 行为模式：有明确的交易目标，行为路径清晰
   - 建议：保持现有用户体验，可考虑推荐相关产品

2. **探索型意图用户（Cluster 2）**
   - 特征：多次页面点击但未完成交易
   - 行为模式：浏览多个功能，行为较为分散
   - 建议：提供更清晰的引导，简化交易流程

### 可视化内容

报告包含以下可视化图表：

1. **用户行为特征对比**：对比不同用户的关键指标
2. **聚类结果（PCA降维）**：在二维空间中展示用户聚类分布
3. **行为模式雷达图**：多维度展示用户行为特征
4. **时间序列分析**：展示用户行为随时间的变化
5. **事件类型分布**：统计不同类型事件的分布情况
6. **意图强度分析**：分析用户的意图强度指标

## 🔧 技术栈

## 📝 算法说明

### 聚类算法

使用K-means聚类算法，通过以下步骤：

1. **特征标准化**：使用StandardScaler对特征进行标准化
2. **降维可视化**：使用PCA将高维特征降至2维进行可视化
3. **聚类分析**：根据用户行为特征进行聚类

### 特征选择

选择的关键特征包括：
- 行为频率特征（事件数、事件密度）
- 行为多样性特征（唯一事件类型、事件多样性）
- 意图强度特征（探索得分、执行得分）
- 行为模式特征（重复行为、行为转换）

## 💡 业务洞察

### 关键发现

1. **完成交易用户**：
   - 会话时长较长，但行为更聚焦
   - 执行得分较高，说明有明确的交易意图
   - 事件密度适中，行为节奏稳定

2. **未完成交易用户**：
   - 事件数较多但会话时长较短
   - 探索得分较高，说明在探索不同功能
   - 重复行为得分较高，可能存在犹豫或困惑

### 优化建议

1. **对于探索型用户**：
   - 提供更清晰的功能引导
   - 简化交易流程，减少操作步骤
   - 增加新手引导和帮助提示

2. **对于高转化用户**：
   - 优化交易流程，提升用户体验
   - 提供个性化推荐
   - 考虑交叉销售机会
